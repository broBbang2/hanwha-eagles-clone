# Hanwha Eagles Clone

## 📌 클론 프로젝트 안내

이 프로젝트는 한화이글스 공식 홈페이지 [https://www.hanwhaeagles.co.kr](https://www.hanwhaeagles.co.kr)을 기반으로 제작된 **클론 코딩 프로젝트**입니다.

- 본 프로젝트는 **개인 포트폴리오** 및 **프론트엔드 학습 목적**으로만 사용됩니다.
- 모든 이미지 및 상표는 해당 저작권자에게 귀속됩니다.


> 요청 시 언제든지 수정/삭제 조치를 취하겠습니다.
